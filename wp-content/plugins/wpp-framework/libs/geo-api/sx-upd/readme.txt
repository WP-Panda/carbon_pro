Скрипт для обновления баз данных Sypex Geo
1. В начале скрипта нужно указать URL для скачивания базы
2. После чего указать каталог в который будет сохраняться dat-файл (по умолчанию в каталог в котором находится sxgeo_update.php)
3. Также можно изменить путь к файлу SxGeo.upd (где хранится дата обновления)

Для запуска в Cron добавить команду

php /полный-путь/sxgeo_update.php
